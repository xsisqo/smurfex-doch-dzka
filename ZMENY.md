# ZMENY – Smurfex Dochádzka

## Kompletný zoznam zmien

### 1. Základ aplikácie
- Vytvorená jednoduchá webová aplikácia typu PWA.
- Aplikácia sa dá nasadiť na GitHub Pages.
- Aplikácia sa dá pridať na plochu mobilu ako bežná aplikácia.

### 2. Logo
- Pridané finálne logo Smurfex, ktoré poslal používateľ.
- Logo je v hlavičke aplikácie.
- Z loga sú vytvorené ikony aplikácie 192x192 a 512x512.

### 3. Jazykové verzie
- Pridaná slovenská verzia.
- Pridaná anglická verzia.
- Pridaný prepínač SK / EN.
- Zvolený jazyk sa ukladá v mobile.

### 4. Dochádzka
- Pridané pole Meno pracovníka.
- Pridané pole Stavba.
- Pridané tlačidlo PRIŠIEL SOM / I ARRIVED.
- Pridané tlačidlo ODCHÁDZAM / I AM LEAVING.
- Aplikácia ukladá dátum, čas, meno, stavbu a typ záznamu.

### 5. Záznamy
- Záznamy sa zobrazujú priamo v aplikácii.
- Záznamy sú uložené lokálne v mobile cez localStorage.
- Pridané tlačidlo Vymazať / Clear.

### 6. Export
- Pridaný export CSV.
- CSV sa dá otvoriť v Exceli, Google Sheets alebo LibreOffice.

### 7. PWA technické súbory
- Pridaný manifest.json.
- Pridaný service worker sw.js.
- Pridané offline cache.
- Pridané theme color a ikony.

### 8. Dizajn
- Modro-biely štýl Smurfex.
- Veľké tlačidlá pre fyzicky pracujúcich ľudí.
- Jednoduchý mobilný vzhľad bez zbytočných funkcií.

## Aktuálny stav
Táto verzia je pripravená na nahratie do GitHub repozitára a nasadenie cez GitHub Pages.
