SMURFEX DOCHÁDZKA PRO - finálny balík na GitHub

Nahrať do GitHub repozitára a prepísať existujúce súbory:
- index.html
- app.js
- style.css
- sw.js

Po nahratí odporúčané:
1. Otvoriť aplikáciu.
2. Spraviť Ctrl + F5 alebo obnoviť stránku.
3. Na mobile prípadne zatvoriť a znovu otvoriť aplikáciu.
