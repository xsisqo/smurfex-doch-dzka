# Smurfex Dochádzka

Jednoduchá PWA aplikácia pre evidenciu príchodu a odchodu pracovníkov Smurfex s.r.o.

## Hotové funkcie
- Logo Smurfex
- Slovenská a anglická verzia
- Prepínač SK / EN
- Meno pracovníka
- Stavba
- Príchod pracovníka
- Odchod pracovníka
- Záznamy uložené lokálne v mobile
- Export CSV
- PWA inštalácia na plochu mobilu
- GitHub Pages podpora
- Service worker a offline cache

## Nasadenie na GitHub Pages
1. Nahraj všetky súbory zo ZIP-u do repozitára.
2. GitHub: Settings → Pages.
3. Source: Deploy from a branch.
4. Branch: main.
5. Folder: / root.
6. Save.
